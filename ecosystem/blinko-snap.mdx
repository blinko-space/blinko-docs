# Blinko Snap Configuration Guide

<img className="rounded-2xl" src="/images/image_1736225286909.png" />

[Blinko Snap](https://github.com/blinko-space/blinko-snap) is a lightweight desktop application built for the Blinko ecosystem that allows you to take notes and capture ideas quickly and easily. This guide will help you understand how to configure and use this tool.

## Basic Setup

### 1. System Tray Operations
<img className="rounded-2xl" src="/images/image_1736225397439.png" />
After installation, Blinko Snap appears as an icon in your system tray. You can control the application through these options:
- Show Window: Display the main window
- Hide Window: Hide the main window
- Settings: Open the settings panel
- Quit: Exit the application

### 2. Configuration Settings
In the settings panel, you need to configure these important parameters:

#### Blinko Endpoint
- Default endpoint: `http://127.0.0.1:1111`
- This is the address of your Blinko server, usually doesn't need modification

#### Blinko Token
- This is your [Access Token](/settings/access-token)
- Used for authentication and connection to Blinko service
- Format example: `eyJhbG...`

### 3. Version Check
- Current version: v0.1.0
- Use the "Check for updates" button to check for new versions

## Usage Guide

### Shortcuts
- Use the global shortcut `Ctrl/Cmd + Shift + Space` to quickly open the note window
- In the note window, you can:
  - Use the toolbar for text formatting
  - Insert images and attachments
  - Quickly save and sync notes

### Toolbar Features
The top toolbar provides various editing options:
- Text formatting (bold, italic, underline, etc.)
- List and indentation control
- Table insertion
- Code block formatting

## Troubleshooting

If you encounter connection issues:
1. Check if the Blinko Endpoint is correct
2. Verify if the Token is valid
3. Ensure the Blinko service is running

## Updates and Maintenance

Regularly check for updates to get new features and bug fixes:
1. Open the settings panel
2. Click "Check for updates"
3. Follow the prompts to complete the update
