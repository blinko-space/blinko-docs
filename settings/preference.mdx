## Theme Settings

<AccordionGroup>
  <Accordion icon="palette" title="Theme Selection" defaultOpen>

    - Toggle using the theme icon in the top right

## Language Settings

<AccordionGroup>
  <Accordion icon="language" title="Interface Language">
    - Select your preferred interface language
    - Changes apply immediately across the application
  </Accordion>
</AccordionGroup>

## Mobile Settings

<AccordionGroup>
  <Accordion icon="mobile" title="Hidden Navigation Bar">
    - Toggle navigation bar visibility on mobile devices
    - When enabled, navigation bar auto-hides while scrolling
    - Improves content viewing experience on smaller screens
  </Accordion>
</AccordionGroup>

## Content Display Settings

<AccordionGroup>
  <Accordion icon="sort" title="Order by Create Time">
    - Toggle between creation time and custom ordering
    - Affects how items are sorted in lists and galleries
    - Enable for chronological organization
  </Accordion>

  <Accordion icon="text-size" title="Text Fold Length">
    - Set maximum character length before text folding
    - Default: 300 characters
    - Helps manage long content display
    - Adjustable based on reading preferences
  </Accordion>

  <Accordion icon="table-cells" title="Device Card Columns">
    - Customize grid layout for different screen sizes
    - Mobile: 1-2 columns
    - Tablet: 2-4 columns
    - Desktop: 2-6 columns
    - Optimizes viewing experience across devices
  </Accordion>

  <Accordion icon="clock" title="Time Format">
    - Choose preferred time display format
    - Customize date display patterns
    - Applies globally across the application
  </Accordion>
</AccordionGroup>

<Note>
Changes to these settings are saved automatically and take effect immediately.
</Note>

<Warning>
Some settings may require a page refresh to fully apply the changes.
</Warning>
