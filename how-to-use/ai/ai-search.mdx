# AI Enhanced Search

<img className="rounded-2xl" src="/images/2024-12-08-14-29-25.png" />

## Overview

AI Enhanced Search is a powerful feature that helps you find information more efficiently. It uses artificial intelligence to understand your search intent and provide more relevant results.

## Key Features

- **Natural Language Understanding**: Search using everyday language instead of exact keywords
- **Smart Filtering**: Automatically filters and ranks results based on [Advanced Setting](/how-to-use/ai/ai-setting#advanced)

## How to Use

1. Click the search bar at the top of the page
2. Type your query in natural language
3. Press Enter or click the search icon
4. Browse through AI-enhanced search results
5. Use the filter options on the right to refine results if needed

## Tips for Better Results

- Be specific in your queries
- Use natural language questions
- Include relevant context in your search
- Try different phrasings if you don't find what you need

## Note

The AI search feature is continuously learning and improving. Your search patterns help make the system more accurate for everyone.
