# 访问令牌配置

访问令牌对于将 Blinko Snap 与您的 Blinko 服务器连接至关重要。本指南将帮助您正确设置和使用访问令牌。

## 入门

### 1. 访问令牌
- 从系统托盘打开 Blinko Snap 设置
- 导航至"设置"部分
- 找到"Blinko 令牌"字段

### 2. 令牌格式
您的访问令牌将采用 JWT 格式，以 `eyJ...` 开头。该令牌用于向 Blinko 服务器验证您的请求。

## 配置步骤

1. **复制令牌**
   - 令牌将显示在设置面板中
   - 仔细复制整个令牌字符串
   - 确保不包含任何额外的空格

2. **验证连接**
   - 输入令牌后，Blinko Snap 将自动测试连接
   - 连接成功将允许您开始使用应用程序
   - 如果连接失败，请验证令牌输入是否正确

## API 测试

如果您想直接测试 API，可以使用以下 cURL 命令：

```bash
curl -X 'POST' 'http://127.0.0.1:1111/api/v1/note/upsert' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer YOUR_TOKEN_HERE' \
-d '{
  "content": "🎉 Hello, Blinko! --send from api",
  "type": 0
}'
```

## 安全最佳实践

1. **令牌保护**
   - 保持令牌安全和私密
   - 不要与他人共享您的令牌
   - 如果怀疑令牌已泄露，重新生成令牌

2. **令牌过期**
   - 令牌可能在一段时间后过期
   - 如果遇到身份验证问题，尝试生成新令牌

## 故障排除

如果遇到问题：
1. 确保完整复制令牌
2. 检查 Blinko 服务器是否正在运行
3. 验证您的网络连接
4. 尝试重新生成令牌

更详细的 API 文档，请访问：`http://127.0.0.1:1111/api-doc`
