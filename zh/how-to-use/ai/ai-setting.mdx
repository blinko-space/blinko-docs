<AccordionGroup>
  <Accordion icon="toggle-on" title="1. 启用AI" defaultOpen>
    - 在设置中找到"**使用AI**"开关。
    - 将其切换到启用状态（紫色）。
  </Accordion>

  <Accordion icon="server" title="2. 选择模型提供商">
    - 从"**模型提供商**"下拉菜单中选择AI模型提供商：
      - 如果使用OpenAI，确保选择"OpenAI"。
      - 如果使用Ollama，确保选择"Ollama"。
  </Accordion>

  <Accordion icon="robot" title="3. 选择AI模型">
    - 从"**AI模型**"下拉菜单中为您的任务选择合适的模型。
  </Accordion>

  <Accordion icon="network-wired" title="4. 设置嵌入模型">
    - 从"**嵌入模型**"下拉菜单中选择嵌入模型。
  </Accordion>

  <Accordion icon="key" title="5. 输入API密钥">
    - 在"**API密钥**"输入框中输入您的OpenAI API密钥：
      - 确保您的密钥正确且具有必要的权限。
  </Accordion>

  <Accordion icon="link" title="6. 设置API端点">
    - 在"**API端点**"输入框中配置API端点：
      - 确保地址有效并与您的API配置匹配。
  </Accordion>

  <Accordion icon="arrows-rotate" title="7. 重建嵌入索引">
    - 如果您需要重建嵌入索引：
      - 点击"**重建**"按钮重新生成嵌入。
      - 如果需要强制重建，点击"**强制重建**"。
  </Accordion>
</AccordionGroup>

<Warning>
确保API密钥和端点的正确性，以避免功能问题。
</Warning>

按照上述步骤，您可以配置并开始使用AI功能。

### 参数解释

<AccordionGroup>
  <Accordion icon="list-ol" title="Top K" defaultOpen>
    - 确定要考虑的最相关片段数量
    - 范围：1-20（推荐：2-5）
    - 更高的值包括更多上下文，但可能降低相关性
  </Accordion>

  <Accordion icon="chart-line" title="分数">
    - 上下文包含的最低相似度分数阈值
    - 范围：0.0-2.0（推荐：0.7-0.9）
    - 更高的值确保更相关的匹配
    - 更低的值包括更多潜在匹配
  </Accordion>

  <Accordion icon="lambda" title="嵌入Lambda">
    - 控制语义相似度和词汇相似度之间的平衡
    - 范围：0.0-1.0（推荐：0.2-0.4）
    - 更高的值倾向于语义含义
    - 更低的值倾向于精确词匹配
  </Accordion>
</AccordionGroup>

### 最佳实践

<Note>
为获得最佳结果：
- 从默认值开始（Top K：2，分数：0.8，Lambda：0.3）
- 根据响应质量逐步调整
- 监控响应相关性和准确性
</Note>

<Warning>
极端参数值可能导致：
- 非常有限或不相关的响应
- 增加处理时间
- 降低准确性
</Warning>
