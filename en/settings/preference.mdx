## Content Display Settings

<AccordionGroup>
  <Accordion icon="sort" title="Order by Create Time">
    - Toggle between creation time and custom ordering
    - Affects how items are sorted in lists and galleries
    - Enable for chronological organization
  </Accordion>

  <Accordion icon="text-size" title="Text Fold Length">
    - Set maximum character length before text folding
    - Default: 500 characters
    - Helps manage long content display
    - Adjustable based on reading preferences
  </Accordion>

  <Accordion icon="table-cells" title="Device Card Columns">
    - Customize grid layout for different screen sizes
    - Mobile: 1-2 columns
    - Tablet: 2-4 columns
    - Desktop: 2-6 columns
    - Optimizes viewing experience across devices
  </Accordion>

  <Accordion icon="clock" title="Time Format">
    - Choose preferred time display format
    - Customize date display patterns
    - Applies globally across the application
  </Accordion>

  <Accordion icon="ruler" title="Max Home Page Width">
    - Set maximum width for the home page layout
    - Enter 0 for maximum available width
    - Helps optimize content display for different screen sizes
  </Accordion>

  <Accordion icon="list" title="Page Size">
    - Set the number of items displayed per page
    - Default: 30 items
    - Customize based on your browsing preferences
  </Accordion>

  <Accordion icon="bars" title="Toolbar Visibility">
    - Customize which toolbar elements are visible
    - Toggle specific tools on/off
    - Personalize your editing experience
  </Accordion>
</AccordionGroup>
